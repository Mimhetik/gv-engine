Crono Expression:
  1st position: seconds

  4th position: day of the month
  5th position: month
  6th position: day of week
  7th position: year

Each element can have the characters following (in addition to the numbers):
  * = all values
  ? = no specific value
  - = specify ranges
  , = additional values
  / = increment
  L = last
  W = weekday
  # = specify the day of month
